slotmach
========
A frustrating slot machine simulator

Controls
========
- `SPACE` to stop slots
- `F` to enter slow-mo
- `K` to turn all slots in one column into *K*s
- `O` to turn all slots in one column into *O*s
- `UP` to expand rows
- `DOWN` to shrink rows
- `LEFT` to expand columns
- `RIGHT` to shrink columns
- `ESC` to quit the program
